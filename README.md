A simple Python script to automate the creation of a powerlifting strength program.

To run, install the `requirements.txt` and then run `python3 ssp.py`

## Program Layout
- `max_sets` define the maximum number of sets per session

### Volume
- Sets: 2 - 5
- Reps: 6 and 12

### Strength
- Sets: 2 - 4
- Reps: 1 - 6
